Aide

Dictionnaire
La recherche peut se faire dans le sens Latin-Français ou Français-Latin.
Latin-Français :
Il est possible d'effectuer une recherche de deux manières :
- en entrant dans la barre de recherche les premières lettres du mot, et en sélectionnant l'entrée correspondante dans la liste déroulante. Le résultat donne la définition, ainsi que les formes dérivées (conjugaisons ou déclinaisons).
- en entrant le mot complet dans la barre de recherche et en pressant sur la touche Recherche du clavier. Le programme effectue alors une analyse morphologique à partir des formes dérivées des entrées du dictionnaire. Le résultat indique les différents résultats possibles.
Français-Latin :
La recherche est effectuée dans le texte des définitions des dictionnaires. Entrez un mot de 3 lettres au moins ; dès que l'application trouve le mot dans les définitions, la liste des résultats s'affiche.

Installation du Gaffiot.
Le célèbre dictionnaire Gaffiot a été numérisé sous formats pdf et base de données par une équipe sous la direction de M. Gérard Greco. Il est possible d'inclure la base de données dans l'application Tabula.
Pour cela, vous devez télécharger le fichier correspondant sur le site de M. Greco (http://gerardgreco.free.fr/), à partir de ce lien : http://gerardgreco.free.fr/spip.php?article47.
Décompressez le fichier .zip, et copiez le fichier ainsi obtenu (au format .tex) dans le dossier Tabula créé par l'application lors de son installation. Ce dossier se trouve sur la carte SD ou la mémoire interne de votre appareil.
Puis, dans l'application, allez dans la partie Textes, et sélectionnez le fichier copié (il se trouvera dans la section Documents). L'installation peut prendre jusqu'à une vingtaine de secondes en fonction de votre appareil.
Si pour une raison quelconque vous souhaitez désinstaller ce dictionnaire, allez dans la partie Dictionnaires de l'onglet Information, et sélectionnez le dictionnaire en effectuant une longue pression.

Textes
La partie Textes comprend plusieurs oeuvres d'auteurs classiques (Salluste, Jules César, Tacite, Saint Augustin, Cornelius Nepos, Saint Jérôme, Virgile) avec le texte original en latin et la traduction française (accessible en touchant le bouton indiquant la langue de lecture).
Il est également possible de charger d'autres textes latins dans l'appplication, en les copiant dans le dossier Tabula.
Les formats de fichier compatibles sont html et pdf
Pour sélectionner un mot dans le texte et effectuer une recherche dans le dictionnaire, sélectionnez-le en touchant simplement l'écran.
Notez que cette fonctionnalité n'est disponible, pour les fichiers pdf, que si ceux-ci contiennent du texte électronique. Les documents scannés ne permettent pas de sélection de texte.
Afin de pouvoir créer le dossier Tabula, l'application doit pouvoir disposer de la permission d'accès en écriture.

Cartes
Vous pouvez créer vos propres listes de cartes mémoires afin d'apprendre de nouveaux mots, à partir des entrées des dictionnaires ou des textes classiques proposés. Pour créer une carte, appuyez sur la croix bleue qui apparaît dans la barre de menu.
Il est possible de paramétrer le nombre maximum de cartes proposées lors d'un test. Une liste d'environ 700 mots est préinstallée, correspondant au vocabulaire latin de base nécessaire pour le collège.
